For using these scripts
Need to enter next fields:

    In reddit_users:
        username - login for reddit
        password - password for redit
        reddit_community - subreddit for posts

    In twitter_users:
        username - login for twitter
        password - password for twitter

    url_for_parse - enter for url from where will be parse xml
    DirForSave - enter full path for directory where will save image for reddit posts
    SeleniumPath - enter full path for selenium webdriver
    proxies - enter proxies of will need else you can left "" - is mean a empty value

In case, when something will happened with posting you can find it in your cron log: "Something happened with posted of reddit" - example

For normal working script need use 1 time per hour as mminimum
